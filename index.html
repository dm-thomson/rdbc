Rate-Dependent Behaviour of Composites (RDBC)¶

This site contains interactive versions of the scatter plots presented in the article titled “A review of the effect of loading rate on the mechanical properties of unidirectional carbon fibre reinforced polymer composites” , published in Composites: Part A, which can also be found in the RDBC GitHub repository along with the rdbc_data.csv file containing the full set of data collected from the literature.
